# Temporal Satellite Super-Resolution

This repository contains a small experiment on temporal super-resolution for satellite imagery.  
Given a short sequence of low-resolution satellite frames, the model learns to reconstruct a sharper, higher-resolution target frame.

## Project overview

The goal is to use temporal information (multiple time steps) to improve spatial detail in satellite images.  
Instead of super-resolving a single image in isolation, this model leverages a neighboring sequence of frames to reconstruct the center frame at higher resolution.

Current status:

- Training and inference pipeline implemented in PyTorch.
- Dataset wrapper for loading short temporal sequences (`sequence_001`, `sequence_002`, …).
- Example trained weights and a qualitative result included in the repo.

## Repository structure

- `src/dataset.py` – dataset class for loading temporal satellite sequences and constructing triplets (context frames and target).[page:0]
- `train_day3.py` – training script that builds the dataset, loads sequences, and optimizes the model.[page:0]
- `infer_sat.py` – inference script for running the trained model on a chosen sequence and saving a visualization.
- `sat_interp_cnn.pth` – example set of trained model weights.[page:0]
- `satellite_result.png` – example output image generated by the model for `sequence_002`.[page:0]

## Data

The code expects a directory of temporal satellite image sequences, each sequence containing multiple frames of the same area over time.  
Each sequence is indexed (e.g., `sequence_001`, `sequence_002`) and the dataset code can be configured to use one or more of these sequences for training and evaluation.[page:0]

## How to run

### Environment

Create and activate a virtual environment (example with Windows PowerShell):

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

(Adjust the commands if you use a different shell or OS.)

### Training

To train the model on `sequence_002`:

```powershell
python train_day3.py
```

This will:

- Load the temporal dataset (with `sequence_002`).
- Train the interpolation/super-resolution network.
- Save updated weights to `sat_interp_cnn.pth`.

### Inference

After training, you can run inference to generate a qualitative result:

```powershell
python infer_sat.py --sequence sequence_002
```

The script will:

- Load the trained model weights.
- Run the model on the chosen sequence.
- Save a visualization to `satellite_result.png`.

You can open `satellite_result.png` to inspect the reconstructed high-resolution frame.

## Results

## Results on sequence_002

For the example frame shown in `outputs/interp_example_day3.png`:

- PSNR (Pred vs GT): 24.79 dB
- SSIM (Pred vs GT): 0.8982

The visualization compares:

- Context frames (t–1, t)
- Ground truth frame at t+1
- Predicted frame at t+1
